Работа с гит

Инициализация

git init

Добавляем изменения локально

 git add .

Добавляем коммит

 git commit -m "test 2 commit"

Пушим

git push -u origin  master

Пушим на гитхаб

git remote add origin  https://github.com/genetik15/git_test.git

Смотрим конфиг

git config --list

Добавляем мэил и логин

        git config --global user.email test@gmail.com
        git config --global user.name test

Скачать с гитхаба

git clone https://github.com/morphIsmail/webforte.git

Хороший плагин для просмотра истории - git history diff

Перезагрузить изменения с гита

git pull
